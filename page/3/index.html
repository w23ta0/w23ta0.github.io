syntax error near unexpected token 的问题

发表于 2015-04-14 • 分类于 Linux/Unix / Shell • 评论: • 阅读次数:
• 字数统计: 147
如果你在windows里写的shell脚本放到linux系统里运行就可以会提示如下错误syntax error near unexpected token 这是因为两个平台下面的换行符不一样导致的。所以需要转换下格式，Notepad++就有这个功能。 首先打开Notepad++,让脚本显示所有符号，发现是CR LF 我们需要将windows的格式转为linux格式，做如下操作即可。 编辑->档案格式转换->转换为UNIX格式 转换之后，发现回车换行符就变了。再放入linux就不会报错了。
